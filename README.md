# Weather-Dashboard

Weather is mostly uncertain. So, before I go about my day, I would like to know what the weather would look like to be better prepared. Weather-Dashboard is a great tool for checking local weather by the minute.

## Description

This weather dashboard application presents local weather to the user at the landing page, and allow the user to find current weather conditions and five days forecast for any cities in the world using https://openweathermap.org/api and https://ipfind.com/ APIs. The site is hosted on GitHub at https://ourosalim-cmd.github.io/Weather-Dashboard/. The code can be found here https://github.com/ourosalim-cmd/Weather-Dashboard


## Installation

This repo consists of front end web files. To run this app locally after cloning the repo, open index.html in your browser of choice. Please keep in mind that both https://openweathermap.org/api and https://ipfind.com/ require personal API keys. The keys used here are free keys that is intended for demonstration purposes. If this key stops working, make  an account athttps://openweathermap.org/api and https://ipfind.com/ to get a free keys and update the related fields in the script.js file.
